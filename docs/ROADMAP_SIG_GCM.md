# SIG-GCM Brasil

## Objetivo

Criar uma plataforma completa para gestão de Guardas Municipais de todo o Brasil.

## Visão

Ser uma plataforma moderna, escalável e multi-município, permitindo que cada Guarda Municipal gerencie suas operações de forma eficiente.

## Princípios

* Multi-município
* Escalas configuráveis
* Aplicativo Android
* Portal do cidadão
* IA integrada
* Dashboard executivo
* Relatórios inteligentes
