# Arquitetura SIG-GCM Brasil

## Objetivo

Criar uma plataforma completa para Guardas Municipais de todo o Brasil.

---

## Conceitos Fundamentais

### Multi-Município

Cada município possui:

* Usuários próprios
* Guardas próprios
* Viaturas próprias
* Escalas próprias
* Ocorrências próprias

Nenhum município acessa dados de outro.

---

### Escalas Configuráveis

Cada município poderá criar:

* 24x72
* 12x36
* Expediente
* Plantão Extra
* Qualquer outro modelo

O sistema se adapta à cidade.

A cidade não se adapta ao sistema.
